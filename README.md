A Python-based command-line application that fetches real-time stock data and visualizes price trends using moving averages.

---

## Features

- Retrieve live stock information using **yfinance**
- Display:
  - Company name
  - Current stock price
  - Daily price change (%)
- Optional **graph visualization**
  - Closing price trend
  - 7-day moving average
  - 30-day moving average
- Simple interactive **CLI interface**

---

## Technologies Used

- Python
- pandas
- yfinance
- matplotlib

---

## Installation & Setup

### 1. Clone the repository
git clone https://github.com/Vaibhav1450/Stock_Analyzer.git
cd Stock_Analyzer

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the program
python stock_price_analyzer.py

---

## Usage

1. Enter a valid stock symbol  
   Examples: `TSLA`, `AAPL`, `INFY`
2. View real-time stock details in the terminal.
3. Choose **"y"** to display the graphical trend.
